How the sheet needs to be set up
  1. The sheet must be viewable by anyone with the link (Share → General access → Anyone with the link → Viewer), or published to the web.
  2. Give the first row headers containing the words run, die, and notes (any order, extra columns are ignored). If no such headers are found, the first three columns are used as Run / Die / Notes.
  3. Each remaining row becomes one label added to the queue.
  4. Paste the normal sheet URL, or a File → Share → Publish to web → CSV link.
